# Instrucciones del repositorio

## Idioma

- Todo comentario, resumen, descripcion de commit, texto de PR, respuesta de review y mensaje dirigido a GitHub debe redactarse en espanol.

## Contexto del proyecto

- Proyecto academico de una clinica llamado `ClinicaPlus`.
- Stack principal: `Next.js`, `TypeScript`, `PostgreSQL`, `Vercel`.
- Arquitectura preferida: monolito SSR simple de mantener.
