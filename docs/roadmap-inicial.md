# Roadmap inicial

## Fase 1: definicion

- confirmar nombre del sistema
- validar modulos principales
- definir tablas y relaciones
- acordar responsabilidades del equipo

## Fase 2: arranque tecnico

- crear proyecto `Next.js` con `TypeScript`
- configurar variables de entorno
- conectar `PostgreSQL`
- preparar despliegue en `Vercel`

## Fase 3: desarrollo base

- login y pagina principal
- esquema de base de datos
- paginas de consulta
- carga de datos de prueba

## Fase 4: reportes

- reporte de citas por doctor
- reporte de pagos y pacientes atendidos

## Fase 5: entrega

- documentacion final
- presentacion
- revision de despliegue
- pruebas funcionales
